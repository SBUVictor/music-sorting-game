Student Information
Key Terms

Whole number
The number after the class.
Examples: M 2, M 2.3, M 3

First cutter
The first letter-number group after the class number.
Examples: .B103, .C4, .S36

Later work element
A short work-style code that comes after the first cutter in some labels.
Example: R4 in M 2.3 .U6 R4 V.66

Year
The publication year.
Example: 1994

Later-line details
Extra details that appear after the main call number parts.
Examples: SER.I, BD.20, PTS.1-6, FASC.3, H.III:E2

How This Version Works
Each module teaches first, gives a worked example, then serves similar practice problems until the learner passes that skill. After the final activity, a score record is prepared for Google Sheets.
