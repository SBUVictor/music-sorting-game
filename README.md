# 🎼 Shelf It Right — Music Score Training

An interactive training tool designed to help student assistants in the Music Library learn how to read and correctly order music score call numbers.

## 📚 Purpose

This tool was created to support the onboarding and training of student workers at the Stony Brook University Music Library. It focuses on developing accuracy and confidence in shelving music scores.

## 🧠 What it teaches

- Understanding M, ML, MT classifications
- Whole number ordering
- Cutter numbers (decimal logic)
- Second Cutters
- Year as a sorting element
- Line-four details:
  - Score types (full score, vocal score, etc.)
  - Parts (violin, cello, etc.)
  - Opus numbers (op.)
  - Numbers within opus (no.)
  - Thematic catalog numbers (K., BWV, Hob.)

## 🎮 How it works

- Each round introduces a concept
- Students answer one question per round
- Immediate feedback reinforces learning
- Difficulty gradually increases
